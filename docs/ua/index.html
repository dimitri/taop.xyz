  • Покращений дизайн книги

    Завдяки схвальним відгукам, які отримало перше видання книги, друге видання тепер доступне у друкованому форматі з м'якою обкладинкою, що ідеально доповнить вашу бібілотеку!

  • SQL - це код

    Дізнайтеся, як працювати з SQL у робочому процесі розробника: версіонування, тестування, перегляд коду, розгортання. Висвітлюються кращі практики та інструменти.

  • Практичні зразки

    Вивчайте розширений SQL за допомогою практичних зразків та наборів даних, що допоможуть вам отримати максимум від книги! Кожен запит вирішує практичний випадок використання і подається у контексті.

  • Моделювання бази даних

    Книга висвітлює (де)нормалізацію за допомогою простих практичних зразків для занурення у цю, здавалося б, складну тему, включаючи стратегію кешування та індексування.

  • SQL для розробників

    Це книга для розробників, що висвітлює передові техніки SQL для обробки даних. Дізнайтеся, як отримати саме той набір результатів, який потрібен у вашому коді додатку!

  • Написання ефективного коду SQL

    Написання ефективного коду SQL простіше, ніж здається, і починається з моделювання бази даних та написання чистого коду. Книга вчить, як писати швидкі запити!

  • З батарейками в комплекті

    Книга йде з відкритими наборами даних та інструкціями, як їх отримати, тож ви можете запускати всі запити SQL вдома, редагувати їх та вчитися на них. У комплект включено навіть розширену установку psql!

    Ця книга - це 438 сторінок, наповнених запитами SQL. 334 з них актуальні, всі вони відповідають нa конкретний зрозумілий приклад використання, що подається простою мовою. Кожен запит детально пояснюється.

    Повний набір для електронного читання на всіх пристроях: планшеті, телефоні та комп'ютері. PDF ретельно редагується відповідно до сучасних стандартів відображення (спробуйте на своєму планшеті!).

The Art of PostgreSQL

Le livre qui enseigne le SQL aux développeurs : apprenez comment remplacer des milliers de lignes de code avec des requêtes SQL simples !

  • Nouveau Chapitre

    Ceci est la seconde édition du livre. Elle inclue un nouveau chapitre au sujet des extensions PostgreSQL. Écrit par l'auteur de la command CREATE EXTENSION!

  • Seconde Édition

    « The Art of PostgreSQL » est le nouveau titre de mon livre « Mastering PostgreSQL in Application Development »/i>, acclamé comme l'une des toutes meilleurs sources existantes pour apprendre le SQL !

  • Un design amélioré

    Après une réception enthousiaste de la première édition du livre, la seconde édition est maintenant disponible avec une finition améliorée. Embellissez votre bibliothèque !

  • Le SQL est du Code

    Apprenez à intégrer le SQL à votre processus de développement : gestion des versions, tests, relecture de code, déploiement. Le livre couvre les « best practices » permettant de travailler efficacement en SQL.

  • Des examples concrets

    Apprenez le SQL avancé à l'aide d'exemples concerts appuyés sur des jeux de données qui aident à tirer le meilleur du livre ! Chacune des nombreuses requêtes SQL du livre est présentée dans le contexte d'une question précise à laquelle répondre à partir des données fournies.

  • Modélisation de base de données

    Le livre couvre les aspects de normalisation des modèles de données avec des exemples concrets et simples. Ceux-ci permettent de plonger dans un sujet qui pourrait être rebutant, et accompagne le lecteur jusqu'aux principes de mise en cache et de stratégie d'indexation.

  • Le SQL pour les développeurs

    Ce livre a été pensé et rédigé pour les développeurs. Nous couvrons des techniques SQL avancées de traitement des données. Apprenez à obtenir précisemment le résultat nécessaire à votre application !

  • Écrire du SQL efficace

    Écrire du SQL efficace est plus facile qu'il n'y parait. Cela commence avec une modélisation de base de données normalisée, étape indispensable à la rédaction de code clair. Le livre enseigne comment écrire des requêtes efficates… et rapides !

  • Tout compris

    Le livre est fourni avec des jeux de données Open Data, et des instructions vous permettant de disposer d'une copie locale des données. Éditez et exécutez le SQL présenté dans le livre afin d'apprendre de manière interactive ! Nous fournissons même une configuration avancée de l'outil psql.

  • Le livre

    Le livre de 438 pages présente 334 reqûetes SQL. Chacune d'entre elle est présentée dans un contexte clair et détaillé en toutes lettres.

  • 6 Interviews

    Des interviews de vétérans de l'industrie sont proposées. Ces intervenants ont commencé à construire les applications webs au siècle dernier. Ils partagent avec nous leurs opinions et leur recul quant à la meilleure façon d'aborder le SQL.

  • PDF, ePub, MOBI

    Les trois formats électroniques sont fournis pour une compatibilité sans faille avec vos différents écrans : tablette, téléphone, ordinateur. Le PDF a été conçu pour être facile à lire sur nos écrans modernes : essayez-le sur votre tablette !

  • Containeur docker

    Une instance PostgreSQL pré-chargée avec les jeux de données présentés dans le livre est mise en avant par une application web interactive qui vous permet de rejouer chaque requête SQL du livre !

  • Jeux de données réels

    Les jeux de données utillisés tout au long du livre représentent 56 tables provenant de différentes sources Open Data. Parce qu'il est plus facile d'apprendre avec des exemples concrets, le livre contient 15 jeux de données bien réels.

  • Partage en groupes de 15 lecteurs

    Partagez le livre et l'instance docker avec vos collègues, jusqu'à 15 personnes par license ! Un seul achat pour toute votre équipe ! Vous disposerez chacun d'une copie des fichiers mis à votre disposition et pourrez avancer à votre rythme.

  • Examples de normalisation

    La modélisation de base de données est bien sûr abordée dans ce livre, et présentée de manière à montrer pourquoi ce sujet est intéressant pour vous, développeur d'application. Les examples concrets de normalisation incluent des scripts et une méthodologie que vous pourrez ré-utiliser dans vos projets.

  • Requêtes SQL & Code

    Toutes les éditions sont fournies avec les 334 requêtes chacune dans un fichier .sql. Vous disposez également des applications exemples écrites pour le livre — en Python, Go, et Java.

  • Tests Unitaires

    Les applications sont fournies avec leurs tests unitaires et de non-régression. Nous utilisons l'outil RegreSQL pour rédiger les tests, et après avoir lu le livre vous voudrez certainement l'utiliserer aussi !

À propos de l'auteur

Bonjour ! Mon nom est Dimitri Fontaine, et j'ai utilisé et contribué à divers logiciels Open Source depuis plus de 20 ans maintenant. Je fais partie des contributeurs majeurs à PostgreSQL après avoir contribué la notion d'EXTENSION ainsi que les EVENT TRIGGERs, entre autres fonctionnalités.

En tant que développeur, j'écris du code de manière quotidienne (ou presque) depuis de nombreuses années, à la fois dans le cadre de mon travail et sur mon temps libre. J'ai été employé comme architecte de bases de données, administrateur de bases de données — d'astreinte ; ou encore développeur d'applications. J'ai toujours apprécié tout particulièrement le SQL. En tant qu'architecte de bases de données, j'ai trouvé agréable de pouvoir accompagner mes collègues dans leur découverte et apprentissage des techniques SQL avancées.

Foire Aux Questions
Et si le livre n'était pas du tout ce que j'en attends ?
« The Art of PostgreSQL » est un livre qui enseigne le SQL aux développeurs d'applications. Si vous vous rendez compte que ce livre ne vous apporte rien, je vous rembourserai votre achat sur simple réception d'un e-mail.
J'ai acheté le livre et je suis confus… quelle est la suite des opérations ?
Le contenu du livre est disponible sous les formats PDF, ePub et MOBI et vous pouvez les télécharger sur le site https://postgresql.thinkific.com/ où vous avez été inscrits en tant qu'étudiant.
Le livre est-il protégé par des DRM?
Non. Le PDF n'est même pas marqué. Vous possédez ce que vous achetez. Les sources de données sélectionnées tout au long du livre sont également disponible sous des licenses de type Open Data, et le livre lui-même est copyright Dimitri Fontaine, tous droits réservés.
J'ai une autre question ?
Vous pouvez me contacter par e-mail à l'adresse dim @ tapoueh.org afin de me poser votre question ! Je ferai de mon mieux pour vous répondre, en général sous 5 jours ouvrés, si je ne suis pas en vacance avec les enfants…
